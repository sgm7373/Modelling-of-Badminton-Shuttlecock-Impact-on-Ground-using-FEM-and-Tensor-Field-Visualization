# Modelling of Badminton Shuttlecock Impact on Ground using Finite Element Analysis and Tensor Field Visualization

## Overview
This project presents a high-fidelity **finite element analysis (FEA)** study of shuttlecock impact dynamics, combined with automated scientific visualization and an immersive **VR-based analysis pipeline**. The work bridges physics-based simulation, data automation, and interactive visualization in a digital-twin-style workflow.

## Key Objectives
- Model and simulate transient impact and rebound behavior using FEM
- Validate numerical accuracy through convergence and solver studies
- Automate extraction and visualization of large-scale simulation data
- Enable immersive analysis of simulation results using virtual reality (VR)

## Methodology
- **Geometry & CAD**: 3D modeling using Blender and Siemens NX  
- **Simulation**: Transient nonlinear FEA using Abaqus/Explicit  
- **Physics**: Contact mechanics, friction, elastic, hyperelastic, and viscoelastic material models  
- **Validation**: Mesh and time-step convergence, implicit vs. explicit solver comparison  
- **Automation**: Python-based pipelines for post-processing and data extraction  
- **Visualization**: Stress, strain, energy, and tensor-field visualization  
- **VR Integration**: Unity-based VR environment for immersive exploration of simulation outputs  

## Features
- High-fidelity impact and rebound simulation
- Automated post-processing of stress, strain, velocity, and energy fields
- Tensor-field visualization for stress propagation analysis
- Interactive VR visualization for deformation and dynamic response
- Reproducible simulation-to-visualization workflow
